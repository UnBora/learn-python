## **1. Line style, color, marker**

```python
import matplotlib.pyplot as plt
import numpy as np

# Generate data
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)

# Plot with different styles
plt.plot(x, y1, color='blue', linestyle='-', marker='', label='sin(x)')   # solid blue line
plt.plot(x, y2, color='red', linestyle='--', marker='o', label='cos(x)')  # dashed red line with circle markers

plt.title("Line Style, Color, Marker Example")
plt.xlabel("X axis")
plt.ylabel("Y axis")
plt.grid(True)
plt.legend()
plt.show()
```

* `color='blue'` â†’ á‡áŸ’ášá¾áŸá–ááŸŒ
* `linestyle='-'` â†’ `'-'` solid, `'--'` dashed, `':'` dotted
* `marker='o'` â†’ `'o'` circle, `'s'` square, `'^'` triangle
* `label='...'` + `plt.legend()` â†’ á”á„áŸ’á á¶á‰ legend

## **3. Subplots: `plt.subplot()`**

* `plt.subplot(nrows, ncols, index)` â†’ á”áŸ‚á„ figure á‡á¶ grid
* Example:

```python
# Subplots example: 2 rows x 2 columns
plt.figure(figsize=(10,6))

# Subplot 1
plt.subplot(2,2,1)  # 2 rows, 2 cols, position 1
plt.plot(x, y1, color='blue')
plt.title('sin(x)')

# Subplot 2
plt.subplot(2,2,2)
plt.plot(x, y2, color='red')
plt.title('cos(x)')

# Subplot 3
plt.subplot(2,2,3)
plt.plot(x, y3, color='green')
plt.title('sin(2x)')

# Subplot 4
plt.subplot(2,2,4)
plt.plot(x, y4, color='purple')
plt.title('cos(0.5x)')

plt.tight_layout()  # Automatically adjust spacing
plt.show()
```

âœ… Comment:

* `plt.subplot(2,2,1)` â†’ grid 2x2, subplot 1
* `plt.tight_layout()` â†’ spacing between subplots
* á˜á¶á“á¢á¶á… customize title, color, linestyle, marker á€áŸ’á“á»á„ subplot á‘á¸á•áŸ’áŸáŸá„áŸ—
