## **1. Scatter Plot (`plt.scatter()`)**

* Scatter plot â†’ plot points individually (x, y)
* Useful to see relationship between two variables

```python
import matplotlib.pyplot as plt
import numpy as np

# Generate random data
x = np.random.rand(50)  # 50 random points for X
y = np.random.rand(50)  # 50 random points for Y

# Scatter plot
plt.scatter(x, y, color='blue', marker='o', s=50)  # s=size of marker
plt.title("Scatter Plot Example")
plt.xlabel("X axis")
plt.ylabel("Y axis")
plt.grid(True)
plt.show()
```

âœ… Comment:

* `marker='o'` â†’ circle, `'s'` â†’ square
* `s=50` â†’ size of points

## **2. Bar Chart (`plt.bar()`)**

* Bar chart â†’ compare quantities

```python
# Categories and values
categories = ['A', 'B', 'C', 'D']
values = [10, 15, 7, 12]

plt.bar(categories, values, color='green')
plt.title("Bar Chart Example")
plt.xlabel("Category")
plt.ylabel("Value")
plt.show()
```

âœ… Comment:

* X-axis â†’ categories

* Y-axis â†’ values

* `plt.bar()` â†’ vertical bar chart

* Horizontal bar chart â†’ `plt.barh(categories, values)`

---

## **3. Histogram (`plt.hist()`)**

* Histogram â†’ visualize distribution of data

```python
# Generate random data
data = np.random.randn(1000)  # normal distribution

plt.hist(data, bins=20, color='orange', edgecolor='black')
plt.title("Histogram Example")
plt.xlabel("Value")
plt.ylabel("Frequency")
plt.show()
```

âœ… Comment:

* `bins=20` â†’ number of bins
* `edgecolor='black'` â†’ draw border for each bar

---

## **4. Pie Chart (`plt.pie()`)**

* Pie chart â†’ show proportion / percentage

```python
sizes = [25, 30, 20, 25]
labels = ['Apples', 'Bananas', 'Cherries', 'Dates']
colors = ['red', 'yellow', 'pink', 'purple']

plt.pie(sizes, labels=labels, colors=colors, autopct='%1.1f%%', startangle=90)
plt.title("Pie Chart Example")
plt.show()
```

âœ… Comment:

* `autopct='%1.1f%%'` â†’ show percentage on each slice
* `startangle=90` â†’ rotate start position

---

## **Summary**

1. **Scatter plot** â†’ individual points
2. **Bar chart** â†’ compare quantities
3. **Histogram** â†’ distribution of values
4. **Pie chart** â†’ proportion of categories
