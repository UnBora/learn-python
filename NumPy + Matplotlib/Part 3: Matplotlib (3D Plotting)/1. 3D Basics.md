## **2. Creating 3D axes**

* ááŸ’ášá¼áœá”á„áŸ’á€á¾á **figure** á“á·á„ **3D axes**
* `projection='3d'` â†’ á”á„áŸ’á á¶á‰ 3D plot

```python
# Create figure
fig = plt.figure(figsize=(8,6))

# Add 3D axes
ax = fig.add_subplot(111, projection='3d')  # 1 row, 1 col, 1st subplot

# Generate some data
x = np.linspace(0, 5, 50)
y = np.sin(x)
z = np.cos(x)

# 3D scatter plot
ax.scatter(x, y, z, color='blue')

# Add labels and title
ax.set_xlabel("X axis")
ax.set_ylabel("Y axis")
ax.set_zlabel("Z axis")
ax.set_title("3D Scatter Example (Basic)")

# Show plot
plt.show()
```

âœ… Comment:

* `fig.add_subplot(111, projection='3d')` â†’ 1x1 grid, subplot 1, 3D projection
* `ax.scatter(x, y, z)` â†’ 3D scatter points
* `ax.set_xlabel`, `ax.set_ylabel`, `ax.set_zlabel` â†’ axis labels
* `ax.set_title()` â†’ plot title
